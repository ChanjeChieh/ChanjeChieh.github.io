解决使用 hexo 博客插入图片插件后图片地址错误的问题

先介绍一种在 hexo博客 中插入图片的方法：

  1. 在 hexo 目录下执行下面的指令来安装这个插件

    npm install hexo-asset-image –save

  2. 修改 hexo 的配置文件 _config.yml

    原：post_asset_folder: false
    修改为：post_asset_folder: true
